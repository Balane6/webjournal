    DATA SCIENCE
    STRUCTURED DATA

    adheres to a per-defined data model and is therefore straightforward to analyze.Structured data conforms to a tabular format with a relationship between the different rows and columns.

    SEMI-STRUCTURED DATA

    a form of structured data that does not comform with the formal structured of data models associated with relational databases or other forms of data tables, but nonetheless, contains tags or other markers to separate semantic elements and enforce hierarchies of records and fields within the data.

    UNSTRUCTURED DATA

    information that either does not have a pedefined data model or is not organized in a pre-defined manner.It is typically text-heavy but may contain data such as dates, numbers, and facts as well.

    Metadata-Data about Data

    it provides additional information about specific set of data

    Data Value Chain

    to describe the information flow within big data system as a series of steps needed to generate value useful insights from data.
